IMPORTANT: Before this application can start, you need to create a Kubernetes secret
with your PostgreSQL DATABASE_URL connection string.

Create the secret with:

  kubectl create secret generic {{ .Values.database.secretName }} \
    --from-literal=DATABASE_URL="postgresql://user:password@host:5432/dbname" \
    -n {{ .Values.namespace }}

Check the deployment status:

  kubectl get deployments -n {{ .Values.namespace }}
  kubectl get pods -n {{ .Values.namespace }}

View logs:

  kubectl logs -n {{ .Values.namespace }} -l app.kubernetes.io/name={{ include "hikma.name" . }}
